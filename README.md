# Long-Read Sequencing Pipeline

This project performs quality control and metric calculation for Nanopore data using Nextflow and Snakemake.

## How to Run
- Snakemake: `snakemake --cores 4`
- Nextflow: `nextflow run main.nf`

## Outputs
Graphs and statistics are located in the `results_snakemake/metrics/` folder.
